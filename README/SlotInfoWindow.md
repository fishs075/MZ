# 今何装備してるの？プラグイン（SlotInfoWindow.js）

このプラグインは、装備画面でアイテム選択ウィンドウが表示されている間、<br>
選択中の装備部位と現在装備しているアイテムを別ウィンドウで表示します。<br>
プラグインを導入して有効化するだけで動作します。<br>
